# How-to-install-KRNL
This will show you how to install krnl executer!

  ## Requirements:
  1. Have Windows **10**
  2. **64** bit OS
  3. Administrator on your PC ( To run and turn off anti virus )
  
  
  
  ## Installing with .exe
  
  1. Turn off your anti virus or else this will not work
  2. Goto **https://krnl.ca/**
  3. Hit the **DOWNLOAD** button
  4. If it says **krnl_console_bootstrapper** is dangerous so we have blocked it then press **ctrl** + **J** and hit keep dangerous file | If your file did not get blocked then skip this
  5. Run the bootstrapper ( if the file disapears when you run it that means your antivirus is still on -.- )
  6. Once finished installing goto the krnl folder
  7. Run krnlss.exe
  8. Done!

## Installing with .rar
1. Turn off your anti virus or else this will not work
2. Make sure you have **Winrar** or **7zip** ( You can get them from here: **https://www.win-rar.com/, https://www.7-zip.org/ **)
3. Download file from **https://wearedevs.net/d/Krnl**
4. If it says **KRNLWRD** is dangerous so we have blocked it then press **ctrl** + **J** and hit keep dangerous file | If your file did not get blocked then skip this
5. Goto your dowloads folder
6. Right click on KRNLWRD.rar
7. Hit extract here
8. Open folder
9. Run krnl.exe
10. Done!
